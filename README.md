# podosome_dynamics
MATLAB is required to run the code. Both scripts execute directly. life_cycle.m plots the quantitative time course of key variables over the oscillatory cycle, while phase_diagram.m generates a three-dimensional phase diagram that maps the rich dynamical landscape of podosome behavior across varying mechanical and biochemical conditions. One can also run biochemical_nosie.m to check the effect of noise on podosome oscillatory behavior and the coefficient of variation (CV) of the oscillation period as a function of noise intensity. To test the effect of noise on phase boundary, one can run coherence_resonance.m and get the coherence resonance due to noise. The stability_analysis_scaling_law.m is used to check the statistical robustness of the mechanosensing scaling law. The asynchronous_synchronous.m is used to show the spatiotemporal evolution of podosome arrays driven by noise. The fisher.m is the one to get the fisher information to show the precision of the podosome mechanosensing on substrate with different stiffness. The waveform_encoding.m is the one to show that podosome oscillations encode substrate stiffness through both waveform shape and proximity to the Hopf bifurcation. The waveform_criticality_efficiency.m is the one to show how the mechanical environment affects the podosomes communication efficiency. The p_myosin_stiffness.m is the one to show how p-myosin affects the podosome oscillatory behavior.
